Bilibili 评论数据集（BLBL-Comment-Dataset）
数据集简介
本数据集包含 B 站多视频的用户评论及关联信息，每条视频对应独立文件夹，内含评论数据、视频元信息等文件，可用于社交网络分析、情感分析、用户行为研究等学术任务，数据均来自 B 站公开评论区，未涉及隐私信息。
数据集结构
根目录结构
数据集根目录以视频 BV 号为独立文件夹命名，每个文件夹对应一个 B 站视频，结构如下：BLBL-Comment-Dataset/├── BV1A2hXzDEnW_探秘废弃的河南省会，这里还剩啥？【地球知识局】/├── BV1A4sJBcEXM_Dory | 坐在地暖上喝冰啤酒.../├── BV1a4wBceBz_【第 234 期】《南海归旧属问题.../└── 更多视频文件夹
单视频文件夹内容
每个视频文件夹内固定包含 4 类文件，具体说明如下：文件名 类型 说明BVxxxx.csv CSV 文件 该视频的用户评论核心数据，包含用户信息、评论内容、互动数据等完整字段BVxxxx.geojson GEOJSON 文件 评论用户地理位置关联数据，部分视频因采集限制可能不包含BVxxxx.html HTML 文件 视频评论区原始网页快照，用于数据溯源和真实性验证video_info.json JSON 文件 视频元信息文件，包含 BV 号、视频标题、UP 主名称、评论总数等基础信息
评论数据字段说明（CSV 文件）
CSV 文件为评论核心数据，包含以下字段，可直接用 Excel、Python 等工具读取：字段名 类型 说明bvid 字符串 视频唯一标识 BV 号，与文件夹名称对应upname 字符串 发布评论的用户昵称，已做基础脱敏处理sex 字符串 用户性别，可选值为男、女、保密content 字符串 评论具体文本内容，保留原始表述（无删减）like 整数 该条评论的获赞数量location 字符串 用户公开填写的地理位置信息level 整数 用户 B 站账号等级（1-6 级）ctime 整数 评论发布时间戳（毫秒级），可转换为标准时间格式

数据来源：采集自 B 站公开视频的评论区，仅获取公开可访问的信息，未爬取用户隐私数据；
使用限制：本数据集仅用于学术研究、课程设计、非商业项目开发，禁止用于商业用途、恶意爬取衍生等违规操作；
责任说明：使用本数据集需遵守 B 站用户协议及相关法律法规，若因违规使用产生纠纷，由使用者自行承担责任。
